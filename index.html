📋 Pipeline Stages — สรุปย่อ

  • Noise: Gaussian / Salt&Pepper / Poisson / Speckle — จำลอง noise ประเภทต่าง ๆ
  • Contrast: Brightness · Gamma · HE / AHE / CLAHE — ปรับการกระจาย intensity
  • Spatial: Unsharp Masking · Laplacian · Kirsch — เพิ่มความคมชัดและตรวจขอบ
  • FFT: LP / HP / High-Boost / Band-Pass / Reject — กรองใน frequency domain
  • Denoise: Gaussian · Median · Mean · Bilateral · Wiener — ลด noise หลังเพิ่ม
  • Segment: Threshold / Otsu / Adaptive · Sobel / Canny / LoG · Region Growing / Watershed
  • Morphology: Erosion · Dilation · Opening · Closing · Thinning · Thickening
  • Pseudo-Color: Hot · Jet · Bone · Viridis — แปลง grayscale เป็นสีเพื่อวิเคราะห์
